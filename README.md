# Blackpill-INAV (FixedWing)
DIY Flight Controller Firmware for Fixed Wing UAVs

* STM32F411CEU6 @ 108Mhz 
* 2x UARTs, 1x Softserial, 2x SPIs, 1x I2C, 2x ADC
* 1x Motor, 7x Servo Outputs
* Blackbox SDCard
* LED_Strip
* PINIO1
* BEEPER

Tested on following sensors/modules:

IMU: MPU6500/BMI160
MAG: HMC5883/QMC5883
BARO: MS5611/BMM280
RangeFinder: VL53L0x
GPS: BN220T
